# text_emotion_detector_python

A simple and interactive Graphical User Interface (GUI) application built with Python that detects emotions from user-inputted text. It leverages the power of the `TextBlob` library for sentiment analysis and provides real-time feedback.

## Features
* **User-Friendly Interface:** Built using Tkinter for ease of use.
* **Instant Detection:** Analyzes text sentiment and identifies underlying emotions.
* **Open Source:** Licensed under MIT, allowing everyone to learn and build upon it.

##  Prerequisites
Ensure you have **Python 3.x** installed on your system. You will also need to install the following dependency:

bash
    pip install textblob


After installing TextBlob, download the necessary data:

bash
    python -m textblob.download_corpora
